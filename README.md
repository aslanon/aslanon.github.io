## Getting Started

1.  Clone the repo:

        git clone git@github.com:aslanon/aslanon.github.io.git

2.  Install dependencies:

        npm install

3.  Run webpack:

        npm start

visit http://localhost:3000

###### \* Dist directory is copied to the root directory to work on the github page
